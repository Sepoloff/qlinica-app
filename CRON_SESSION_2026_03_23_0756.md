## 📝 Notes for Future Sessions

### Memory Leak Prevention
The new `SafeAsyncCache` with TTL is now the recommended pattern for any caching:
```typescript
const cache = new SafeAsyncCache<string, Data>(5 * 60 * 1000); // 5 min TTL
const data = await cache.getOrFetch('key', () => fetchData());
```

### Form Pattern Best Practice
Use `createFormState()` for new forms instead of useState chains:
```typescript
const form = createFormState(
  { email: '', password: '' },
  async (values) => await api.login(values),
  { successMessage: 'Logged in!' }
);
```

### Booking Window Validation
Always use `isWithinBookingWindow()` with clinic-specific constraints:
```typescript
const available = isWithinBookingWindow(selectedDate, {
  minDaysInAdvance: 1,  // Can't book today
  maxDaysInAdvance: 90, // Can't book too far ahead
});
```
